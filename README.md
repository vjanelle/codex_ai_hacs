# Codex AI for Home Assistant

Custom Home Assistant integration that uses Codex-style ChatGPT device auth and
the Codex backend for AI Tasks, Assist conversation, STT, and TTS.

## Install

Copy `custom_components/codex_ai` into your Home Assistant `custom_components`
directory, or install this repository as a HACS custom integration repository.

## Status

Experimental. This uses Codex's ChatGPT backend auth pattern rather than a
standard OpenAI API key.
